===========
A+B Problem
===========
:Problem ID: 1000
:Time Limit: 1 S
:Memory Limit: 128 MB

Problem
-------
Input two number *a* and *b*

Output *a + b*

Examples
--------
0. Input: ::

     1 1

   Output: ::

     2
1. Input: ::

     15 5

   Output: ::

     20

Data Range
----------
:math:`a+b<=2^31-1`
